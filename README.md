# 🚩 Topic

- 맛있게 먹었던 음식들을 장소와 함께 일기로 남기는 장소 기반 일기 서비스
- 또는 장소를 검색하여 폴더에 저장함으로써 맛집 버킷 리스트 등으로 활용 가능

# 🪧 Summary

기획자 또는 디자이너와 함께 프로젝트를 만들어가는 서비스 SWYG 3기로써 참여하여 기획자분과 협업하여 만든 서비스 입니다. <br />
SWYG 홈페이지에 정식으로 배포되어 누구든지 이용할 수 있습니다.
([링크](https://www.swygbro.com/contents))

가고 싶은 곳을 지도에 저장할 수 있습니다. 또한 일기를 쓰듯이 방문한 장소와 음식에 대한 기록을 남길 수 있습니다.  <br />
기록을 공개해 정보를 공유하는 사이트로도 활용할 수 있습니다.

# 🛣️ How to Execute

```
// 프로젝트 복제
git clone https://github.com/seongjin2427/food-diary.git

// 서버 실행
npm run dev
```

# 🔑 Key Function

- **일기 작성**
    - **위지윅 tiptap 라이브러리를 사용하여 사용자가 일기를 필요에 따라 편집하며 작성 가능**
    - **사진을 포함하여 작성 가능**
- **일기 수정 및 삭제**
    - **기존에 작성했던 일기를 수정하거나 삭제 할 수 있음**
- **일기 검색**
    - **작성된 일기를 날짜 필터 및 검색 단어를 토대로 찾을 수 있음**
- **장소 검색**
    - **Kakao Map API를 이용, 검색단어와 관련된 장소들을 찾을 수 있음**
    - **장소 검색 페이지에서 검색 시, 해당 장소가 어디에 위치하는지 지도를 통해 확인할 수 있음**
- **폴더 기능**
    - **검색한 장소에 대해 폴더를 지정하여 필요 시 확인할 수 있음**
